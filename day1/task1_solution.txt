# High Availability (HA) based Web UI (Frontend Webapp)

## Infrastructure Requirements

### Virtual Machines
- Minimum VM requirement: 2 VMs
- Deployment: Across 2 zones in selected region

### Operating System
- Platform: Linux
- Distribution options:
    - Ubuntu
    - CentOS

### Application Server
- Selection criteria needed for UI deployment
- Must support HA configuration

### Notes
- Implementation requires HA-based application architecture
